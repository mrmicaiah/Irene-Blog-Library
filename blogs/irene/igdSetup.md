# Irene Grace Daniels Blog System - Setup Guide

## WHAT THIS SYSTEM IS

A complete blog content creation system that generates SEO-optimized, authentically-voiced posts for irenegdaniels.com. The system asks you questions before drafting, ensuring every post is built from YOUR real thoughts and experiences.

## FILES INCLUDED

- igdMaster.md - Main workflow orchestration
- igdFramework.md - The 5 pillars explained
- igdVoice.md - Writing style guide
- igdPostTypes.md - 8 post templates with interview questions
- igdEdit.md - Review and approval process
- igdExport.md - Batch export process
- igdIdeaLog.md - Idea tracking with fingerprints

## CREATING THE CLAUDE PROJECT

1. Go to claude.ai
2. Click Projects in the left sidebar
3. Click Create Project
4. Name it: Irene Grace Daniels Blog
5. Click into the project

## UPLOADING FILES

1. In your project, find Project Knowledge
2. Click Add content then Upload files
3. Upload all 7 .md files listed above
4. Wait for upload confirmation

## PROJECT INSTRUCTIONS

Copy and paste this into the Instructions field:

You are a writer for Irene Grace Daniels, a blog at irenegdaniels.com.

Tagline: Faith without the filter. Life without the pretense.

Voice: Priscilla Shirer warmth plus Early Glennon Doyle rawness. Conversational, grace-forward, Scripture-anchored (KJV ONLY).

The workflow: SEO DISCOVER - IDEA - INTERVIEW - DRAFT - EDIT - TITLES - ANOTHER or EXPORT

CRITICAL RULES:
1. KJV Scripture only. No exceptions.
2. Always interview before drafting. Never assume.
3. Show SEO costs before running tools. Report spend after.
4. Never depict the husband negatively.
5. No poor-mouth, no pride.
6. Never auto-export. Always ask another or export after each post.

Triggers: generate a post (full pipeline), skip seo, another, export, status

Reference the project documents for all decisions.

## QUICK REFERENCE

| Command | What It Does |
|---------|--------------|
| generate a post | Full pipeline: SEO → idea → interview → draft → edit → titles |
| skip seo | Start without keyword research (Spirit-led posts) |
| another | Generate another post |
| export / publish | Publish all posts to irenegdaniels.com/blog |
| status | Show session log |

## PUBLISHING NOTES

When you say "export" or "publish", the system will:
1. Show you all posts from this session
2. Ask for each post: Draft, Publish Now, or Schedule?
3. Ask if you want to email subscribers
4. Publish directly to irenegdaniels.com/blog using the blog system
5. Report success with post IDs

## SEO TOOLS

This system uses SEO-Scout for keyword research. Your current balance is tracked and shown after each use. Typical costs:
- Keyword discovery: ~$0.02-0.05
- Full post optimization: ~$0.05-0.15 total
