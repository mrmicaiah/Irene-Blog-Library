# Irene Grace Daniels Edit Process
## Review and Approval Checklist

---

## OVERVIEW

Every post goes through this edit review before titles are generated. The goal is to ensure voice consistency, structural integrity, SEO optimization, and Scripture accuracy.

---

## EDIT CHECKLIST

### 1. VOICE CHECK

| Question | Pass/Fail |
|----------|-----------|
| Does this sound like Irene talking to a friend? | |
| Is it warm but not saccharine? | |
| Is it honest but not exhibitionist? | |
| Are there any preachy or lecturing moments? | |
| Is grace the undertone, even when truth is hard? | |
| Any poor-mouth or pride creeping in? | |

**Phrase check:**
- Did we use "here is the thing" more than once? Rotate.
- Did we use "listen" more than once? Rotate.
- Any overused words? (just, should, amazing, incredible)

### 2. STRUCTURE CHECK

| Question | Pass/Fail |
|----------|-----------|
| Does the opening hook draw the reader in? | |
| Is the post type structure followed? | |
| Does each section earn its length? | |
| Is there a clear through-line from start to finish? | |
| Does the ending land with weight? | |
| Is the word count appropriate for the post type? | |

**Word count targets:**
- Short posts: 600-1,000 words
- Long posts: 1,500-2,500 words

### 3. SCRIPTURE CHECK

| Question | Pass/Fail |
|----------|-----------|
| Is all Scripture KJV? (NO EXCEPTIONS) | |
| Are references complete? (Book, chapter, verse) | |
| Is the verse quoted accurately? | |
| Does the Scripture earn its place or just decorate? | |
| Is there space for the verse to breathe? | |

### 4. SEO CHECK

| Question | Pass/Fail |
|----------|-----------|
| Is the target keyword in the first 100 words? | |
| Does the keyword appear naturally 3-5 times? | |
| Is there a keyword in at least one subheading? | |
| Does the content match search intent? | |
| Is keyword density natural, not forced? | |

### 5. BOUNDARY CHECK

| Question | Pass/Fail |
|----------|-----------|
| Does this avoid depicting the husband negatively? | |
| Is the separation mentioned without blame? | |
| No victim language or poor-mouth tone? | |
| No pride or braggy tone? | |
| Would Irene be comfortable with this being public? | |

---

## EDIT REPORT FORMAT

After running the checklist, present findings showing:
- Overall status (APPROVED or NEEDS REVISION)
- Each category status
- Specific issues to address with suggested fixes
- Strengths identified
- Confirmation to proceed to titles

---

## COMMON ISSUES AND FIXES

### Voice Issues
- Too preachy: Add personal struggle, use "I" more than "you should"
- Too vague: Add specific details, name the thing
- Too heavy: Add a moment of lightness or hope
- Repetitive phrases: Swap for alternatives from rotation list

### Structure Issues
- Weak opening: Start with scene, question, or confession
- Saggy middle: Cut filler, every paragraph must earn its place
- Abrupt ending: Add the "so what" - the invitation or takeaway
- Too long: Cut the parts that repeat what you already said

### Scripture Issues
- Wrong translation: Replace with KJV immediately
- Missing reference: Add book, chapter, verse
- Decorative verse: Either explore it more or cut it

### SEO Issues
- Keyword missing from intro: Work it into the first paragraph naturally
- Keyword stuffed: Remove some instances, keep it natural
- No keyword in headings: Add to at least one subheading

---

## APPROVAL FLOW

1. Run full checklist
2. Present edit report
3. If APPROVED: proceed to titles
4. If NEEDS REVISION: make changes, run checklist again
5. Get Irene approval before generating titles
