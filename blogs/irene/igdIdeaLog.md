## Tracking System with Fingerprints

## FINGERPRINT FORMAT

Each fingerprint is a 7-character code:
- Position 1: Pillar (R=Real, W=Word, G=Grace, B=Behind, L=Light)
- Position 2: Life Stage (T=Teen, S=Single, W=Wife, M=Mom, R=Trenches, P=Stepmom, E=Empty, C=Crisis, O=Starting Over, A=All)
- Position 3: Topic (1-9 and A-F for 15 topics)
- Position 4: Post Type (1-8)
- Position 5: Tone (W=Warm, G=Gentle, H=Honest, J=Joyful, R=Reflective, F=Fire)
- Position 6: Scripture (P=Phil4:8, O=Other, G=General, N=None)
- Position 7: Gap (M=Marriage, O=Motherhood, F=Faith, B=Body, I=Financial, L=Timeline, R=Relationship, S=Self, C=Calling, N=None)

## TOPIC KEY
1=Marriage, 2=Motherhood, 3=Singleness, 4=Faith, 5=Identity, 6=Body, 7=Finances, 8=Relationships, 9=Grief, A=Fear, B=Shame, C=Loneliness, D=Purpose, E=Writing, F=Home
